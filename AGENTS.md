# Project Rules

- This is a paper-trading research agent.
- Live trading is forbidden in Phase 1.
- Exchange API private keys are forbidden in Phase 1.
- All important behavior must have tests.
- Run `pytest` after each meaningful code change.
- Strategy changes must modify exactly one allowed variable.
- Strategy versions must be saved before modification.
- Prefer simple Python.
- Keep the project runnable on Linux.
- Never commit secrets.
- Never modify files outside the repository.
